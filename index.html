My projects

Over the years, I have made a lot of personal projects, mostly games, websites, or a combination of both.

Most of these projects were made as a way for me to learn about the technology, so they might not be useful, popular or even functional, but they are definitely fun.

Here are a list of my best ones so far.

TAG

TAG is a numeric puzzle game where players try to solve levels using as little moves as possible. It was made for PC and Android devices.

This project taught me a lot about game development (level design, animation, data management, etc.) and how to use Unity Game Engine to make one.

Get the game here.

ISLE

ISLE is a web game, emulating a study environment while giving users a sense of progress the longer they spend focusing.

This is my first large scale project, having everything from persistent data storage in a NoSQL database, website hosting to account login and authentication. The game is also continously updated and regularly receives user input for bugs or features.

Try out the game here.
